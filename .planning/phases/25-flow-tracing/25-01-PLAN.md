<feature>
  <name>traceRoute: shortest path formatting and validation</name>
  <files>src/search/trace.ts, tests/search/trace.test.ts</files>
  <behavior>
    --- Response shape ---
    - traceRoute(db, "app-api", "app-payments") for a 2-hop path returns { from: "app-api", to: "app-payments", path_summary: "app-api -[grpc]-> app-gateway -[http]-> app-payments", hop_count: 2, hops: [{from: "app-api", to: "app-gateway", mechanism: "grpc"}, {from: "app-gateway", to: "app-payments", mechanism: "http"}] }
    - hops contain only from, to, mechanism (and optionally via for event/kafka)
    - No confidence fields anywhere in TraceResult or TraceHop
    - No repo IDs in response -- names only

    --- Arrow chain path_summary ---
    - grpc/http/gateway hops: "A -[grpc]-> B" (no via)
    - event hops with via: "A -[event: OrderCreated]-> B" (mechanism: via format)
    - kafka hops with via: "A -[kafka: orders-topic]-> B"
    - event hops without via (null): "A -[event]-> B" (just mechanism, no colon)

    --- via field ---
    - via included in hop ONLY for event/kafka mechanism AND when non-null
    - grpc/http/gateway hops: via field omitted entirely from hop object
    - event/kafka hops with null via: via field omitted

    --- Same-service ---
    - traceRoute(db, "app-api", "app-api") returns { from: "app-api", to: "app-api", path_summary: "app-api (same service)", hop_count: 0, hops: [] }
    - Checked BEFORE calling shortestPath

    --- Error: service not found ---
    - One missing: throws Error("Service not found: app-foo")
    - Both missing: throws Error("Services not found: app-foo, app-bar")
    - Validates BOTH upfront, reports both in one error

    --- Error: no path ---
    - throws Error("No path between app-api and app-foo")

    --- Confidence ---
    - TRACE-04 simplified: no confidence fields. The TraceHop type has no confidence property. The TraceResult has no min_confidence property.
  </behavior>
  <implementation>
    Follow the exact pattern from src/search/impact.ts:
    1. Import buildGraph, shortestPath from ./graph.js
    2. Define TraceHop interface: { from: string, to: string, mechanism: string, via?: string }
    3. Define TraceResult interface: { from: string, to: string, path_summary: string, hop_count: number, hops: TraceHop[] }
    4. traceRoute(db, from, to) function:
       - Build graph with buildGraph(db)
       - Validate both services via graph.repoIds.get() -- collect missing into array, throw once
       - Check from === to early, return zero-hop success
       - Call shortestPath(graph, fromId, toId)
       - If null, throw no-path error
       - Map GraphHop[] to TraceHop[] (strip IDs, strip confidence, conditionally include via)
       - Build path_summary with buildPathSummary helper
    5. buildPathSummary(hops) helper: builds arrow chain from hop list
    6. Export traceRoute, TraceResult, TraceHop
  </implementation>
</feature>
