       ```typescript
       const jsonMode = opts.json || !process.stdout.isTTY;

         printSummary(process.stdout, results, elapsedMs, errors);
       }
       ```

    4. Move the `errors.printSummary(process.stderr)` call: In JSON mode, keep errors on stderr (current behavior). In human mode, `printSummary` already handles error output to stdout via ErrorCollector delegation, so remove the separate stderr call to avoid duplicate error printing. Specifically:

       ```typescript
       // Remove the existing standalone error print block:

         errors.printSummary(process.stderr);
       }
       // In human mode, printSummary() handles it via the stream parameter
       ```

    5. Keep `if (opts.timing) reportTimings();` at the end (unchanged).
