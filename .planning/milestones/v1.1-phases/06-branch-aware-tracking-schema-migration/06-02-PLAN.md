<objective>
Refactor the indexing pipeline and all extractors to read from the default branch via git plumbing commands instead of the filesystem working tree.

Purpose: This is the core behavioral change of Phase 6. After this, `kb index` always indexes main/master content regardless of which branch is checked out locally. Repos without main/master are skipped with a warning. The `default_branch` value is persisted to the repos table for audit/debugging.

Output: Updated pipeline.ts orchestrating branch-aware reads, refactored extractors accepting content from pipeline, updated metadata.ts reading from branch, writer.ts persisting default_branch, and integration tests proving the feature branch isolation behavior.
</objective>
